# Sensor-Based Vehicle Controller (Unity)

A physics-based autonomous vehicle system developed in Unity using C#,  
focused on **raycast-based sensing, obstacle detection, and smooth vehicle control**.

This project demonstrates clean gameplay logic, sensor integration, and real-time
decision-making without using external AI libraries.
